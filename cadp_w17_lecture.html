開啟Leo 步驟1

  1. 到https://copy.com/zpYyuIcuB3fHAdr0 (fix.zip)下載
  2. 開啟Ana3_lite2→data→SciTE→SciTE.exe
  3. 用SciTE.exe打開下載的fix資料夾裡的fix.bat和start.bat

  2. 回到SciTE.exe
  3. 將fix.bat裡面多一行copy leo-editor-broke-abbrev.zip C:\Ana3_2015\data\→儲存
  4. 將start.bat裡面第35行改為pip install leo-editor-broke-abbrev.zip→儲存
  5. 全部修改完回去點start後就成功了

自評分數:65

跟著老師的步驟一項一項做，最後開啟Leo，過程不會很複雜，但是中間還是有一些部份比較不容易懂(如:更改python的東西)要經過老師的講解才比較了解。而製作投影片的過程，則是對照班上同學的投影片製作出來的，再加上之前的步驟，讓我更上手。
